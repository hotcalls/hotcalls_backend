Was ist schiefgelaufen?
Passwort-Reset-Links können aus verschiedenen Gründen fehlschlagen:
  • Der Link ist abgelaufen (nur 24 Stunden gültig)
  • Der Link wurde bereits verwendet
  • Der Link wurde beim Kopieren beschädigt
  • Ihr Kontostatus hat sich geändert

Lösung: Fordern Sie einen neuen Passwort-Reset-Link an und versuchen Sie es erneut.
